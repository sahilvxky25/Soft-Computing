# Soft-Computing

This repository contains MATLAB Live Scripts (`.mlx`) that demonstrate core soft-computing and optimization techniques used in machine learning and intelligent systems.

## Contents

- `Fuzzy Logic.mlx` — fundamentals and examples of fuzzy logic systems.
- `Genetic Algorithm Optimization.mlx` — optimization using a genetic algorithm.
- `MLP.mlx` — implementation/demo of a multilayer perceptron.
- `Netwon Method.mlx` — optimization using Newton's method.
- `PCA.mlx` — principal component analysis workflow.
- `Particle Swarn Optimization.mlx` — optimization using particle swarm optimization.
- `Perception Training.mlx` — perceptron training procedure.
- `Simple Perception.mlx` — basic perceptron example.
- `Steepest Descent.mlx` — optimization using steepest descent.

## Requirements

- MATLAB with support for Live Scripts (`.mlx`)
- Basic understanding of machine learning and optimization concepts

## How to Use

1. Open MATLAB.
2. Clone or download this repository.
3. Open any `.mlx` file from the repository in MATLAB.
4. Run sections interactively to follow each algorithm step by step.

## Notes

- File names are kept as they exist in the repository.
- Scripts are educational and can be used as references for lab work and practice.
